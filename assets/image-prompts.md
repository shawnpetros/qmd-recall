# QMD Recall image prompts

Prompts for ChatGPT image model v2 / `gpt-image-2` assets. These are designed for README, launch posts, and social cards.

## Brand notes

- Product: QMD Recall
- Category: OpenClaw plugin, deterministic memory retrieval
- Feel: fast, precise, calm, technical, not cyberpunk soup
- Avoid: robot faces, generic glowing brains, LinkedIn SaaS gradients, busy dashboards
- Motifs: indexed cards, citations, a clean claw mark, wiki pages, retrieval beam, stopwatch

## README hero

```text
Create a clean 16:9 README hero image for an open source developer tool called “QMD Recall”. Visual metaphor: a fast deterministic search layer pulling three cited snippets from a tidy personal wiki into an AI prompt. Minimal technical aesthetic, dark background, crisp white text, subtle blue and amber highlights, small citation cards with fake paths like memory/2026-04-28.md#L12. Include the title “QMD Recall” and subtitle “Deterministic memory retrieval for OpenClaw”. No robot faces, no generic brain, no em dashes. High legibility, polished GitHub README style.
```

## Social card

Create a 1200x630 social preview card for “QMD Recall”, an OpenClaw plugin. Show a clean pipeline: User message -> QMD search -> cited memory snippets -> assistant prompt.
